## Quick Start

### Install server dependencies

```bash
npm install
```

### Install client dependencies

```bash
cd client
npm install
```
### Run Client Server

```bash
cd client
npm start
```

### Run Express Server

```bash
nodemon
or
node server.js
```
or
### Run both Express & React from root

```bash
npm run dev
```
